Sou estudante de **Análise e Desenvolvimento de Sistemas** com foco em criar aplicações escaláveis que resolvem problemas reais. Atualmente, desenvolvo soluções Full-Stack utilizando **Node.js, PHP, Prisma e React**, sempre priorizando código limpo e a melhor experiência para o usuário final.

Tenho forte experiência com infraestrutura de servidores Linux (Ubuntu/Docker) e configurações avançadas de redes e DNS via Cloudflare, o que me permite ter uma visão 360º dos projetos. Também possuo domínio em pipelines de deploy, utilizando o Git de forma essencial para o build e gerenciamento de dependências no Netlify. Estou sempre aprimorando minhas habilidades através de plataformas como Cisco Networking Academy, Alura e DIO.

## 🚀 Projetos em destaque

💈 **NewPlay Manager (SaaS):** Plataforma robusta para gerenciamento de clientes e serviços. Inclui dashboard financeiro completo com métricas de gastos, lucros e controle de automação de faturamento para IPTV/VPN.
*Tech: React, Node.js, Prisma, PostgreSQL*

💇 **Barber Shop SaaS:** Sistema multi-role (Admin, Secretária, Barbeiro e Cliente) para gestão de barbearias, com controle de filas, histórico de pagamentos e lógica de repasses.
*Tech: React, Node.js*

🏋️ **Descomplicando o Movimento:** Plataforma SaaS focada em personal trainers. Gerenciador avançado de avaliações físicas com monitoramento de fadiga, sono e automação para geração de relatórios em PDF.
*Tech: React, Node.js*

💰 **Máfia Conecta (Controle Financeiro):** Sistema de gestão focado no controle financeiro da Máfia Conecta. Inclui organização de fluxo de caixa, monitoramento de despesas e receitas de forma intuitiva.
*Tech: React, Node.js, Prisma*
